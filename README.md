## 💬 Firebase Chat Uygulaması
Bu proje, React ve Firebase kullanılarak geliştirilmiş, gerçek zamanlı mesajlaşma (chat) uygulamasıdır. Kullanıcılar giriş yaptıktan sonra anlık olarak mesaj gönderip alabilir. Arayüz tasarımı SCSS (Sass) ile modern bir görünüme kavuşturulmuştur.

- 🚀 Özellikler
- 🔐 Firebase Authentication ile kullanıcı girişi
- 💬 Gerçek zamanlı mesajlaşma (Firestore kullanılarak)
- 🧑‍🎨 SCSS ile responsive, sade arayüz
- 🔐 Oturum açma ve çıkış yapma işlemleri
